내가 가진 파이썬 코드에서 GRU 입력 feature가 지금 너무 단순합니다.

현재 feature는 아래 2개입니다.
- Disparity
- MA_Momentum

여기에 아래 4개 feature를 추가하고 싶습니다.
- Return_1d  = (Close / Close.shift(1) - 1) * 100
- Return_5d  = (Close / Close.shift(5) - 1) * 100
- Return_20d = (Close / Close.shift(20) - 1) * 100
- Return_60d = (Close / Close.shift(60) - 1) * 100

내 요구사항:
1. 기존 코드 구조는 최대한 유지할 것
2. preprocess_data() 안에서 위 수익률 feature들을 생성할 것
3. prepare_sequences()에서 feature 목록을 기존 2개에서 6개로 변경할 것
4. input_dim도 자동으로 맞게 수정할 것
5. 기존 타겟(Target_Return)은 그대로 유지할 것
6. 데이터 누수(leakage)는 만들지 말 것
7. 결측치 처리(dropna)는 새 feature 추가 후에도 정상 동작하게 할 것
8. 전체 실행 가능한 완성 코드로 다시 작성할 것
9. 수정된 부분에는 주석으로 "추가 feature"라고 표시할 것

출력 형식:
- 설명 길게 하지 말고
- 전체 실행 가능한 최종 코드만 출력
- 중간 생략 없이 한 번에 완성본으로 출력
